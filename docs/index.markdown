# Conversational-BI

{: .attention}
> **_This project is meant to be a reference architecture and implementation demonstrating the use of Generative AI (GenAI) for decision support analytics in the retail space. Rather than following the traditional reporting lifcycle model, this architecture proposes using LLMs to generate both SQL based on the data warehouse schema to calculate KPIs, and Javascript D3 visualizations to present them._**
